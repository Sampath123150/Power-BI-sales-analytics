# Power-BI-sales-analytics

Power BI dashboard project analyzing sales, profit, customers, and regional performance using interactive visualizations and data storytelling techniques. Includes insights, KPIs, slicers, and trend analysis for better business decision-making.
# Project Overview

This project focuses on Data Visualization and Storytelling, where the objective is to build insightful dashboards that reveal meaningful business patterns using Power BI. The dataset used is Superstore Sales Data, and the analysis highlights performance across categories, states, regions, discounts, and profitability.

The final output consists of interactive dashboards along with story-driven insights that help business stakeholders make informed decisions.

# Objectives

Transform raw sales data into clear business insights

Build interactive and visually appealing dashboards

Identify hidden patterns in sales, profit, and discount behavior

Enable slicer-based filtering for better data exploration

Present insights in a storytelling format to simplify decision-making 

# Tools Used

Power BI Desktop

Power BI visualizations (Bar chart, Donut chart, Waterfall chart, Tree map, Area chart, Cards)

Slicers and filters for interactivity
